# HTML/CSS Version

This is the pure HTML, CSS, and vanilla JavaScript implementation of the landing page.

## Features

- Pure HTML5 semantic markup
- CSS3 with CSS Grid and Flexbox
- Vanilla JavaScript (no frameworks)
- Responsive design
- FakeStore API integration

## How to Run

### Option 1: Open directly
Simply open `index.html` in your web browser.

### Option 2: Use a local server (recommended)

**Using Python:**
```bash
python -m http.server 8000
```

**Using Node.js:**
```bash
npx serve
```

**Using VS Code:**
Install the "Live Server" extension and click "Go Live"

Then visit: `http://localhost:8000`

## Files

- `index.html` - Main HTML structure with SEO meta tags
- `styles.css` - All styling including responsive design
- `script.js` - API integration and dynamic content rendering
